Popis Projektu

Tento projekt je zaměřen na správu a analýzu experimentálních dat shromážděných z různých vědeckých studií. Databáze obsahuje celkově 6 tabulek (viz. obr. níže). Vazba M:N je mezi tabulkou scientist a experiments. Jako povinně volitelné funkce projektu byly vybrány využití poddotazu (subquery) a zobrazení obrázku, který není v souboru, ale přímo v BLOBu v databázi. V kódu jsou použity parametrizované dotazy (prepare a bind_param), což pomáhá zabránit SQL injection útokům tím, že odděluje data od SQL dotazů a nepřímo zakazuje vkládání SQL kódu do uživatelských vstupů.


Tento web obsahuje celkově 4 webové stránky:

  • Domů - obsahuje uložené prvky do databáze
  • Přidat data - umožňuje přidat data do databáze pomocí formulářů
  • Readme - info o projektu
  • Report - agregační dotaz mezi tabulkami scientist a experiments (celkový počet experimentů pro každého vědce); využití poddotazu (subquery); zobrazení obrázku, který není v souboru, ale přímo v BLOBu v databázi
